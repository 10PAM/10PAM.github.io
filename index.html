Clockwork Universe

This is a project that I had worked on during my Senior year of High School, and is one of the most up to date showcase I have of video game development. In this, I used the Roblox Studio game engine, and I focused on memory optimization by ensuring lower polies to enhance performance, making use of client replication to optimize memory usage on the back-end, limiting nested logic statements when not needed to be nested, adding comments to my scripts to ensure ease of finding certain functions in case of needing to go and add, renove, or fix something, adding debounces to prevent excessive inputs, setting up cooldowns to towers as far as allowing a target to be damaged only once every other few seconds which can be seen with the "Flame Thrower" Trap while still allowing other targets to be damaged, using loops only when needed and on the client rather than on the server, and ensuring that player inputs were as bug free as possible. The key takeaway from this project that I had was memory optimization, ensuring fault tolerant logic, reusing functions to keep code small and concise, and most importantley improving my time management skills.

Concept: Tower Defense against Waves of Skeletons
